What we hope to achieve

We aim to make a better more efficient defence against rockets and air related attack.

our goal is to make a machine that tracks object movment and then if the object moves faster then our thresh-hold it will fire the laser and track the object.

for our proof of concept we will operate on a smaller scale using an esp32 camera hooked up to a firebase server which will send the video the camera is streaming

to an object movment detector that will determine whether there was object movment and will calculate thier speed to see if it triggers the thresh-hold.
